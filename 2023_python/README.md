# 2023_python

### Some commands:

**Enter virtual env**
```bash
make venv
```

**Run tests**
```bash
make tests
make tests ARGS=-v
make coverage
```

**Run some day's puzzle**

For example, run first day's puzzle 1.
```bash
python3 -B src/day1/run_puzzle1.py
```
