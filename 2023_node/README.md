# 2023_node

### Some commands:

__Run tests__
```bash
npm test
```

__Run one specific test__
For example run  day 1, puzzle 2's tests
```bash
npm test -- test/day1/puzzle2.test.ts
```

__Build__
```bash
npm run build
```

__Run some day's puzzle__
For example, run first day's puzzle 1.
```bash
node dist/day1/runPuzzle1.js
```
